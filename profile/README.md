## What we build

**Lofa** is a full-stack commerce platform that gives Tunisian artisan brands their first proper online storefront, and gives Tunisian shoppers a curated home for **Made in Tunisia** fashion, crafts and beauty — with cash-on-delivery anywhere in the country.

| Surface | Stack | Status |
| --- | --- | --- |
| Web storefront | Next.js 14 · Tailwind · Vercel | Live · [lofa.tn](https://lofa.tn) |
| Mobile app | Expo / React Native · NativeWind | Beta |
| API | NestJS · Prisma · Postgres · Railway | Live · [api.lofa.tn](https://api.lofa.tn/health) |

## Highlights

- 🇹🇳 **Local-first** — every brand on Lofa is Tunisian-owned and Tunisian-made.
- 💵 **COD across Tunisia** — pay at delivery, no card required.
- ⚡ **Fast** — sub-second LCP on desktop, optimized for 3G/4G mobile.
- 🛡️ **Secure** — strict CSP, HSTS preload, Trusted Types, weekly Dependabot.
